# Price Scraper & Data Analyzer

A Python-based project for extracting electronic component data and visualizing price differences.

## Features
- **Scraper**: Extracts titles and prices from HTML structures.
- **CSV Storage**: Saves processed data into a structured CSV file.
- **Analysis**: Generates a bar chart to compare component prices using Matplotlib.

## How to run
1. Install requirements: `pip install pandas matplotlib beautifulsoup4`
2. Run scraper: `python scraper.py`
3. Run analyzer: `python analyzer.py`
